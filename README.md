## 🖼️ What is AgentVista?

AgentVista lets you test programs that see and act in complex visual situations. It focuses on real-world challenges where software must use multiple tools over time to complete tasks. If you want to see how these smart programs work in tough image scenes, AgentVista gives you the platform to try them out. It does not require programming knowledge to get started.

## 🛠️ Using AgentVista

AgentVista focuses on benchmarking, which means testing how well different visual agents perform in hard scenarios.

### How to Run Tests

1. On the main screen, choose an agent from the list.
2. Select the test scenario you want to run. Each scenario includes images, tool use challenges, and task goals.
3. Click the “Start” button.
4. Watch as the agent attempts to solve the task.
5. When the test ends, results show performance details like accuracy, time, and tool use steps.

### Understanding the Results

- **Accuracy** shows how correct the agent’s actions were.
- **Time Taken** is how long the agent needed.
- **Tools Used** lists the tools invoked by the agent during the task.
- Review these to compare multiple agents or test your own against challenging visual problems.
